Interface

Dashboard

Real-Time Overview: Displays live system metrics, including local disk storage usage, active file counts, and a live log of the background queue workers.

The Visual Queue

A visual grid showing the real-time status of every job. It clearly distinguishes between Pending (Waiting), Image/Final Processing (Active), and Completed states.

Album Management

The high-level hub for organizing datasets. Users can create distinct Albums (e.g., "February Images") to keep different sorting logic isolated.

Inside the Album (Upload & Categories)

The workspace for a specific album. This interface combines the drag-and-drop upload zone with a view of the "Smart Categories" (like Animals, Hand Writings, etc.) that the AI will sort files into.

Category View

The final destination. This shows the "Arts" category, confirming that the AI successfully identified and routed the drawing and pixel art images into this specific folder.

File Details (AI Logic)

A deep dive into a single processed file (pixel.jpg). It reveals the raw JSON data, showing the AI's "Confidence Score" and the text-based "Reasoning" it used to classify the image.
